# String Manager

String Manager handles all your localization needs — from key discovery to AI-assisted translation to profile publishing — without leaving your workflow.

## What you get

- **Profiles** — locale + environment groupings (e.g. `en:prod`, `fr:staging`)
- **Label keys** — auto-discovered from your codebase via CLI or MCP scan
- **Drafts** — translation proposals (human or AI) that go through a review step before publishing
- **Publish** — push a profile to Cloudflare KV for instant edge delivery

## How it works

1. Run `shipeasy i18n scan` (or use the MCP tool) to discover keys from your source code
2. Create a translation draft — manually or via AI
3. Review and approve the draft
4. Publish the profile
