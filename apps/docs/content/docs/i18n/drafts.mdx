# Drafts

A draft is a set of proposed translations for a profile. Drafts can be created manually or generated by AI, then reviewed and merged before publishing.

## Draft lifecycle

```
open → (review) → merged
              └→ abandoned
```

- **Open** — draft is pending review
- **Merged** — translations have been promoted to the profile
- **Abandoned** — draft was discarded without merging

## Creating a draft

Via CLI:

```bash
shipeasy i18n translate --profile fr:prod --ai
```

Via dashboard: **String Manager → Drafts → New draft**

## AI translation

When using `--ai`, the CLI calls your configured LLM to generate translations for all keys missing in the target profile. The AI uses the source (English) values and any context annotations you've added.

## Reviewing a draft

Open the draft in the dashboard to review each key's proposed translation. Edit inline, then click **Merge** to promote to the profile.

## Publishing after merge

Merging a draft does not automatically publish. Run `shipeasy i18n publish` or click **Publish** in the profile view to push to KV.
