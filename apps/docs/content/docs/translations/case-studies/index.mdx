## Translate one button [#first-key]

> _Stage-A stub._

```tsx
import { t } from "@shipeasy/sdk/server";
<button>{t("cta.subscribe", { default: "Subscribe" })}</button>;
```

```bash
shipeasy i18n push --keys cta.subscribe
shipeasy i18n translate --locales fr,de,ja
```
