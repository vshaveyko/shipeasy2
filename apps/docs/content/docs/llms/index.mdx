# LLM Guides

These guides are written to be consumed by AI coding assistants (Claude Code, Codex, OpenCode, Cursor, etc.) and executed autonomously. Each guide covers a complete onboarding workflow for one ShipEasy product.

**How to use:** Point your AI assistant at one of the URLs below and say _"follow this guide"_. The guide tells the agent exactly what to install, what commands to run, and where to stop for human input.

## Guides

- [**String Manager (i18n)**](/llms/i18n-strings) — Install the MCP server and SDK, scan the codebase, run the codemod, push keys, and publish. One stop for auth.
