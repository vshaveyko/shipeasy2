title: API reference
description: REST endpoints for bug reports and feature requests.
---

The feedback API lives under `/api/admin/bugs/*` and `/api/admin/feature-requests/*`. Same auth
model: `Authorization: Bearer sdk_admin_…` + `X-Project-Id: <projectId>`.

```bash
curl https://shipeasy.ai/api/admin/bugs \
  -H "Authorization: Bearer sdk_admin_..." \
  -H "X-Project-Id: prj_..."
```

## Resources

<TileGrid>
  <Tile href="#bugs" icon="◉" title="Bugs">

  <Tile href="#feature-requests" icon="✦" title="Feature requests">

    per IP.
  </Tile>
</TileGrid>

## Bug payload

```json
{
  "title": "Checkout button overlaps footer on iOS Safari",
  "pageUrl": "https://example.com/checkout",
  "stackTrace": "TypeError: ...",   // optional
  "screenshotUrl": "https://...",  // optional
  "userId": "u_abc"                 // optional, server-derived from session if omitted
}
```

## Feature request payload

```json
{
  "title": "Dark mode for the dashboard",
  "importance": "important",
  "userId": "u_abc"
}
```
